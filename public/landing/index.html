    THE PROBLEM

    슬랙으로 인터뷰 녹취 던지고, 스프레드시트로 발화 정리하고, 키노트로 리포트 다시 짜는 일.
    이젠 한 페이지에서 끝납니다.

    — 리서처 8인의 워크플로우를 1년간 관찰하며 만든 도구
    HOW IT WORKS

    3분이면 첫 산출물.
    그 다음부터는 흐름이 됩니다.

    각 단계의 결과는 워크스페이스에 자동 등록 → 다음 도구로 드래그.

    STEP 01

    업로드 / 붙여넣기

    녹취 파일, 인터뷰 노트, 설문 CSV, 또는 한 줄짜리 주제. 어떤 입력이든 시작점이 됩니다.

    →
    STEP 02

    도구를 잇는다

    전사록 → 인터뷰 결과 → 정량 분석. 사이드바의 8개 도구를 결과 → 입력으로 연결.

    →
    STEP 03

    리포트로 닫는다

    워크스페이스의 모든 산출물을 Full Report로 엮어 클라이언트에게 그대로 전달.

    리서치 데이터는 어디에 저장되나요?+

    Supabase Postgres + Row-Level Security. 데이터는 조직 단위로 격리되며, 같은 조직 안에서 역할(`owner / admin / member / viewer`)에 따라 접근이 결정됩니다. 각 산출물은 `private / org / shared` 가시성을 가집니다.

    한국어 인터뷰도 잘 처리되나요?+

    한국 리서처를 1순위로 만든 도구입니다. 화자 분리, 존대/반말 정제, 한국어 인용문 추출 모두 한국어 코퍼스로 검증됩니다. 영어는 같은 모델로 자동 처리됩니다.

    크레딧은 만료되나요?+

    아니요. 구매한 크레딧은 만료되지 않으며, 팀 단위로 공유됩니다. 가입 직후 24시간 동안은 모든 도구가 크레딧 차감 없이 무제한으로 열립니다.

    다른 도구(Slack, Notion)와 연결되나요?+

    현재는 워크스페이스 내부에서 산출물을 다음 도구의 입력으로 연결하는 데 집중하고 있습니다. Slack/Notion export는 2026 Q3 로드맵.

    오픈AI나 다른 모델을 직접 가져올 수 있나요?+

    Enterprise 플랜에서 BYO Key를 지원합니다. 기본 플랜은 자체 모델 라우팅으로 운영되어 사용자가 모델 선택을 신경쓰지 않아도 됩니다.
