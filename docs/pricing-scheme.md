- **포지셔닝 = A (저렴 · 대중 접근).** 기존 "프리미엄 컨설팅급"(₩150k/데스크리포트)에서, **개인 연구자가 부담 없이 만질 수 있는 툴**로 재포지셔닝.

- **스킴 = 하이브리드.** 크레딧(미터링 단위) + 구독(MRR · 편의) + 무료 grant(획득). 크레딧 **단가는 무할인** — 할인은 75% 불변식을 깨므로 금지(§3.3).

## 2. 스킴 개요 (한눈에)

| 축 | 역할 | 값 |
|---|---|---|
| **크레딧** | 모든 위젯 사용의 미터링 단위 | ₩500/cr **고정 · 무할인** |
| **일회성 팩** | 크레딧 선구매 (수량만 차이, 단가 동일) | Mini~Max, ₩500/cr |
| **구독** | MRR · 편의(무만료 · 우선처리 · 시트) | Solo/Plus/Pro, 포함 cr 도 ₩500/cr 환산 |

모든 결제 경로(팩 · 구독 · grant)에서 **매출/크레딧 = ₩500 을 유지**하는 것이 핵심 — 이래야 75% 불변식이 경로와 무관하게 성립합니다(§3.3).

## 3. 75% 마진 불변식 (핵심 공식)

### 3.1 공식 유도

결제수수료 6% 가정. 순마진

```
m = 0.94 − COGS/매출  ≥  0.75
  ⇒ COGS/매출 ≤ 0.19
  ⇒ 매출 ≥ COGS / 0.19
```

₩500/cr 에서 매출 = 500 × cr 이므로:

```
500 × cr ≥ COGS / 0.19
  ⇒ cr ≥ COGS / (500 × 0.19)
  ⇒ cr ≥ COGS / 95
  ⇒ 위젯 최소 크레딧 = ceil( COGS(₩) / 95 )
```

> **한 줄 규칙:** `minCredits[widget] = ceil( COGS(₩) / 95 )`.

### 3.2 코드 강제 (D1 가드)

- 위젯별 `minCredits` 상수를 `src/lib/features.ts` 에 박습니다.
- CI 가드(D1)로 **모든 위젯에 대해 `FEATURE_COSTS[k] ≥ minCredits[k]`** 를 강제 — 위반 시 머지 차단.
- 이렇게 하면 누군가 실수로 위젯 가격을 floor 아래로 내려도 CI 가 잡습니다.

### 3.3 무할인 원칙 (왜 할인이 금지인가)

크레딧 단가를 **어떤 경로에서도 할인하지 않음**으로써 매출/크레딧 = ₩500 이 항상 유지됩니다. 팩을 대량 구매하든, 구독에 포함되든, grant 로 받든 크레딧 1개의 매출 귀속은 동일합니다.

→ 만약 "300cr 팩은 20% 할인" 같은 볼륨 디스카운트를 두면 그 경로의 매출/cr 이 ₩400 으로 떨어지고, COGS 가 그대로면 순마진이 75% 아래로 깨질 수 있습니다. **그래서 팩은 수량만 다르고 단가는 전부 ₩500 입니다.** 구매 유인은 "할인" 이 아니라 **구독의 편의(무만료 · 우선처리 · 시트)** 로 제공합니다(§5).

|---|--:|--:|--:|---|

| 리포트 | 1,500 | 16 | 50 | ✅ 여유 |

> 위 COGS 는 대부분 **추정치**입니다. 실측이 들어오면 이 표와 `features.ts` 의 `minCredits` 를 함께 갱신하세요. min-cr 이 현행 cr 을 넘어서는 위젯(현재는 없음, 통역만 경계)은 즉시 가격 상향 대상입니다.

### 4.1 통역 실측 근거 (2026-07-13)

- 통역 prod 사용량: **~7 오디오-시간 / 6주**(저볼륨).
- OpenAI 전체 비용: **$30 / 2주** — 전액을 통역에 귀속한다고 극단 가정해도 **$13/hr 상한**.
- → 통역 순마진 ≈ **72–81%**, 정확히 **75% 경계선**.
- → 이 경계선을 사람 판단에 맡기지 않고 **E1(실 오디오-분 과금 + cr floor)** 로 못박습니다(§6).

## 6. 통역 가드레일 (E1)

통역은 §4.1 실측상 75% 경계선이라 **두 겹의 가드**로 마진을 자동 보장합니다.

1. **실 오디오-분 과금** (wall-clock 아님).
   - `translate_recordings.duration_sec` 기반으로 **실제 오디오 길이**만 과금.
   - 좀비 세션 · 침묵 구간으로 새는 매출·원가를 정렬(원가는 실 오디오에 비례, 과금도 실 오디오에 비례).

2. **분당 cr floor** = `ceil(COGS_실측 / 95)`.
   - 실측 ≤ $8.5/hr 이면 현행(125cr/hr 수준) 유지.
   - 실측이 초과하면 floor 로 **자동 상향**(상한 ~180cr/hr = $13/hr 대응).
   - §3.2 의 D1 가드가 통역에도 동일하게 적용됩니다.
